# Italy Residential SPV — Pilot Deal

**Working name:** Italy Home Club (Deal #1)  
**Status:** Exploration / structuration (pre-acquisition)  
**Objective:** Execute a first test deal (< €500k) and establish a replicable structure for a portfolio.

---

## What this is

A Canada–Italy residential real estate project: Canadian investors (mostly individuals) invest in Italian value-add residential assets. Execution is led on the ground in Italy by the operator (Sam), with local partners (construction, materials). One Italian S.r.l. (SPV) per property for risk isolation and scalability.

**Strategy:** *Lifestyle-first asset, value-add, flexible exit* — premium STR/hybrid primary; flip as Plan B. Decision gate at 12–24 months: Keep / Sell / Refinance.
