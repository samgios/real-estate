# Interactive tools — Italy Deal #1

Open these HTML files **in a browser** (double-click or drag into Chrome/Safari/Firefox). No server required; they work offline. Data for the scorecard and checklist is embedded; the checklist saves progress in `localStorage` for this browser.

| File | Purpose |
|------|--------|
| **region-scorecard.html** | Sort regions by column (click header); filter by STR demand, prefab viability, €500k feasible. |
| **budget-calculator.html** | Enter purchase + reno + transaction % and FX rate → total EUR/CAD; optional ADR + occupancy → simple gross yield. |
| **compliance-checklist.html** | Italy (pre-launch) and Canada (ongoing) checklists; state saved in browser until you clear. |

Full list of visual tools (including static Mermaid diagrams and tables) is in [../00-Visual-Tools.md](../00-Visual-Tools.md).
